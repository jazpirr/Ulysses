# Ulysses: **A Promise to Your Future Self.**
Ulysses is an Android application designed to help you regain control of your digital life. Inspired by the ancient myth where Ulysses tied himself to his ship's mast to resist the Sirens' call, this app allows you to make a binding pact with your future self by locking away distracting apps and tracking your usage.

## âœ¨ Features

* **Activity Tracking Dashboard:** Monitor your digital habits with precise analytics. Track total screen time, notifications, device unlocks, and view week-by-week statistics of open attempts and prevention rates for specific apps.
* **The Ulysses Pact (Timeout):** Select distracting apps and lock them away for a custom duration (1 Day, 3 Days, 1 Week, 1 Month, or a specific calendar date and time). 
* **Final Commitment Engine:** Once a lock is set and the "Final Commitment" is accepted, the decision is ironclad. The app prevents impulsive changes, ensuring you follow through on your self-imposed screen time limits until the timer expires.
* **Integrated Productivity Tools:** Beyond just app blocking, Ulysses includes built-in Notes and Alarm functionalities right from the bottom navigation bar to keep your workflow centralized.
* **Emergency Access:** A built-in emergency bypass ensures safety is never compromised while you are locked out of your standard apps.

## ðŸš€ Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing.

### Prerequisites

* [Android Studio](https://developer.android.com/studio) (Latest version recommended)
* Java Development Kit (JDK) 11 or higher
* Git

### Installation & Setup

1. **Clone the repository**
   ```bash
   git clone [https://github.com/JIVERU/Ulysses.git](https://github.com/JIVERU/Ulysses.git)
   cd Ulysses
   ```
2. Open the Project
   - Launch Android Studio.
   - Select File > Open and navigate to the cloned Ulysses directory.
   - Allow Gradle to sync and download the necessary dependencies.

4. Build and Run
   - Connect an Android device via USB (with USB Debugging enabled) or start an Android Emulator.
   - Click the Run 'app' button in Android Studio (Shift + F10).
   - Alternatively, build to an apk and download in a mobile device.
